sign-up

提交：邮箱、用户名、密码、确认密码、身份

返回：0/1/2

if：

​	用户名不包含非法字符且不超过长度限制

​	密码不包含非法字符且不超过长度限制

​	密码与确认密码相同

​	身份至少选择一项，不包含实验室官方

then：返回1

else if：身份至少选择一项，包含实验室官方

then：返回2

else：返回0



点击阅读服务协议按钮

if：返回1或2

then：打开服务协议页面

else：弹出报错提示框	



点击我已阅读服务协议并同意

if：返回1

then：

​	弹出成功提示框

​	（1s后）跳转到main

else if：返回2

then：

​	弹出成功提示框

​	（1s后）跳转到main

​	发送身份审核申请（记入数据库）
